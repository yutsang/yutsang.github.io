# Insights tab

- **URL:** `/insights/`
- **Source:** `insights/index.html` (Jekyll `layout: page`).
- **Posts:** Add Markdown files in **`_posts/`** at the repository root (`YYYY-MM-DD-title.md`). They are not stored inside this folder—Jekyll collects `_posts/` globally.
- **Styling:** Jekyll theme CSS from `style.scss` → `/style.css`. Empty-state and list styles live in `_sass/_layout.scss` (`.insights-*`).
