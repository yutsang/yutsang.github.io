# Site tabs ↔ repo folders

| Tab (header) | Public URL | Primary folder / files |
|--------------|------------|-------------------------|
| **Home** | `/` (`index.html`) | Repo root: `index.html`, `css/site.css`, `css/about.css`, `timeline-data.js`, `js/*`, `home/README.md` |
| **Insights** | `/insights/` | `insights/index.html` (Jekyll), `_posts/*.md` (content), `style.scss` / `_sass/` (Jekyll theme) |
| **Data Viz** | `/data-viz/` | `data-viz/index.html` |
| **Projects** | `/projects/` | `projects/index.html` |
| **Me** | `/contact/` | `contact/index.html` |

Older subsites such as `apps/` and `dashboards/` still exist; prefer the URLs in the table for the main experience.

Redirects at repo root (`about.html`, `projects.html`, `data-viz.html`, `contact.html`, `apps.html`) keep old bookmarks working.
