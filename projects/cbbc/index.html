CBBC Street Map牛熊證街貨地圖

Where the bulls and bears are parked — rebuilt every trading morning from HKEX public data 牛熊街貨聚集何處——每個交易日早上由 HKEX 公開數據自動重建

§ 01 / The idea概念

Reading the street.解讀街貨。

Every callable bull/bear contract (CBBC) has a call level: touch it and the product is called mandatorily. Where outstanding positions cluster — the street — therefore maps where leveraged money is betting, and where a move would trigger cascades of forced calls. Issuer sites show this chart but keep the data; here it's computed independently from HKEX's public end-of-day files — call levels, entitlement ratios, and outstanding quantities for every listed CBBC. 每隻牛熊證都有收回價：一觸及即被強制收回。所以街貨（未平倉街外持貨）喺邊個價位聚集，就標示咗槓桿資金賭緊邊度——亦標示咗指數行到邊度會觸發連環強制收回。發行商網站有呢幅圖但唔開放數據；呢度由 HKEX 公開嘅收市後檔案獨立計算——全市場每隻牛熊證嘅收回價、換股比率同街貨量。

A GitHub Action fetches the file each trading morning (outstanding quantities refresh ~06:15 HKT), aggregates street size by call-level zone in index/share-equivalent units, and commits a compact JSON that this page renders. No backend, no API keys — the same self-updating pattern as the rrag wiki graph. GitHub Action 每個交易日早上（街貨數據約 06:15 HKT 更新後）自動抓取檔案，按收回價區間聚合成等價指數／股份單位嘅街貨量，commit 一個精簡 JSON 由本頁渲染。無後端、無 API key——同 rrag 知識網絡圖一樣嘅自我更新模式。
