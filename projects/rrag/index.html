§ 01 / The idea概念

An LLM as librarian.讓 LLM 做圖書管理員

rrag implements Karpathy's LLM Wiki pattern: raw sources — course notes, papers, articles — go into an Obsidian vault, and a Claude agent compiles them into a structured, cross-referenced wiki. The agent summarizes, links related pages with [[wikilinks]], rates its own confidence per page, and keeps the index healthy through /ingest, /query, and /lint workflows. rrag 實作 Karpathy 的 LLM Wiki pattern：raw sources——課堂筆記、論文、文章——放入 Obsidian vault，由 Claude agent 編纂成結構化、互相引用的 wiki。Agent 負責摘要、以 [[wikilinks]] 連結相關頁面、為每頁標注 confidence 評級，並透過 /ingest、/query、/lint 等 workflows 維持索引健康。

The graph below is the wiki's actual link structure, rebuilt weekly by a GitHub Action that parses every page. Hubs like optimization, linear-programming, and financial-ratios emerge on their own — nobody planned the taxonomy; it grew out of what got ingested. 下方的 graph 是 wiki 真實的連結結構，由 GitHub Action 每週解析所有頁面自動重建。optimization、linear-programming、financial-ratios 這些 hub 是自然浮現的——沒有人預先設計分類，它由被 ingest 的內容自己長出來。
